# ToDoList-App
 ToDo App using HTML, CSS & Vanilla JavaScript
 -Implemented IIFE, fetch API, Revealing Module Pattern, Event Listners, filter on Array, etc.

- Functionality
	- Add a TODO
	- Delete a TODO
	- check task
	- total items count
    - Fetch ToDo List

- Data
	- tasks - an array
	- task - {done, text, id}

- Functions (in code)
    - fetchToDos
    - addTaskToDOM
    - renderList
    - toggleTask
    - deleteTask
    - addTask
    - showNotification
    - handleInputKeypress
    - handleClickListener
    - initializeApp
